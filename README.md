# closed-loop-power-control-system
Simulation of a closed-loop DC power distribution system with PID-based voltage regulation, transient response analysis, and fault injection testing.
## Features
- First-order RC power stage model
- PID voltage controller
- Closed-loop simulation
- Transient response visualization

## Engineering Concepts Demonstrated
- Control systems theory
- Time-domain response analysis
- Stability and regulation
- Dynamic system modeling
